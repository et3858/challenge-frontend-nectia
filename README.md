# Nectia challenge

This is only a project as a part of nectia's challenge. It uses React in Vite.




## Install modules and dependencies

```sh
npm install
```

## Run in dev mode

```sh
npm run dev
```

## Compile and run in prod mode

```sh
npm run build && npm run preview
```

## Available paths

- `/`
- `/login`
- `/home`
